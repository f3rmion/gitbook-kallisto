# Need Help

### Getting Help

Besides this manual, you can check the in-program help by

```bash
> kallisto --help
```

### The Silent Mode

You can get rid of the output by using the silent mode, which is usable via the `--silent` flag.

```bash
> kallisto --silent option argument
```
